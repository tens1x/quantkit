# QuantKit 使用指南

## 启动

```bash
uv run python -m quantkit
```

## 第一步：导入你的真实持仓

主菜单选 `4` Portfolio → `1` Import CSV → 输入文件路径

支持两种格式：
- **IBKR 导出文件**：直接导入 Interactive Brokers 的 Transaction History，自动识别，仅导入买入交易
- **QuantKit CSV**：`symbol,buy_date,buy_price,quantity,market`

导入后选 `2` View positions 确认持仓。

## 第二步：Factor Check — 给持仓做"体检"

主菜单选 `1` Factor Check → `2` All positions

每只股票会得到 6 个维度的评分：

| 因子 | 问的问题 | 怎么看 |
|------|---------|--------|
| PE (市盈率) | 现在贵不贵？ | 股价相对盈利是否偏高 |
| PB (市净率) | 资产打了几折？ | 股价相对净资产的溢价 |
| ROE (净资产收益率) | 公司赚钱效率高不高？ | >15% 算优秀 |
| Revenue Growth | 营收在增长吗？ | 公司是在扩张还是萎缩 |
| Volatility (60d) | 这只股票颠不颠？ | 波动大 = 风险大，但也可能有机会 |
| Momentum (20d) | 最近趋势如何？ | 涨太猛可能是追高，跌太多可能是恐慌 |

**红绿灯怎么读：**
- 🟢 绿灯：健康区间
- 🟡 黄灯：需要关注
- 🔴 红灯：风险信号

**重要：红灯不代表要卖。** 比如 NVDA 可能 PE 很高（红灯），但如果你看好 AI 长期增长，高 PE 可能是合理的。关键是你要**有意识地**做判断，而不是糊里糊涂持有。

每次看到红灯，问自己：**"我知道这个风险吗？我接受吗？为什么？"**

## 第三步：Strategy Backtest — 验证你的"直觉"

主菜单选 `2` Strategy Backtest

### 可用策略

| 策略 | 逻辑 | 适合验证什么 |
|------|------|-------------|
| MA Cross (均线交叉) | 短期均线上穿长期均线买入，下穿卖出 | "追涨杀跌到底赚不赚钱？" |
| DCA (定投) | 每月固定日买入固定金额 | "不看盘、无脑定投效果如何？" |
| Low PE (低估值) | PE 低于历史分位数买入 | 暂未支持，敬请期待 |

### 结果怎么看

| 指标 | 含义 | 参考 |
|------|------|------|
| Total Return | 策略总收益 | 正数 = 赚了 |
| vs Buy & Hold | 比"买了不动"好多少 | 负数 = 折腾还不如不动 |
| Max Drawdown | 最大回撤 | -20% 意味着 100 万可能变 80 万 |
| Win Rate | 胜率 | 50% 以上算及格 |
| Sharpe Ratio | 风险调整后收益 | >1 不错, >2 很好 |
| Trade Count | 交易次数 | 次数多不代表赚得多 |

### 学习要点

1. **大部分散户策略跑不赢 Buy & Hold。** 如果 vs Buy & Hold 是负数，说明你折腾半天还不如买了不动。这是最重要的认知——**交易越多不代表赚越多**。

2. **同一策略在不同股票上效果差异巨大。** 多试几只就会发现：没有万能策略。

3. **看 Max Drawdown，不只看收益。** 一个策略年化 30% 但最大回撤 -50%，你真的拿得住吗？

## 第四步：Risk Lens — 看清全局风险

主菜单选 `3` Risk Lens，自动分析你的整个组合。

### 1. 集中度 (Concentration)

某只股票占仓位超过 30% 会标红。问自己：如果它暴跌 30%，你的总资产会怎样？你能接受吗？

### 2. 相关性 (Correlation)

如果你买了 NVDA 和 STX（都是科技/半导体），它们相关性可能很高（>0.7 标红）。涨一起涨、跌一起跌——你以为分散了，其实没有。

真正的分散是买入**相关性低**的资产。

### 3. 波动贡献 (Volatility Contribution)

哪只股票对组合波动影响最大？可能仓位不大但波动剧烈的股票，才是你组合风险的主要来源。

### 4. 最大回撤 (Max Drawdown)

"基于历史数据，你的组合最多从高点跌多少。" 这个数字就是你需要做好的心理准备。

## 学习循环

建议每月做一次：

```
导入最新持仓
    ↓
Factor 体检 → 问自己"我为什么持有这只？理由还成立吗？"
    ↓
Backtest 验证 → "我的交易策略真的有效吗？"
    ↓
Risk Lens 全局 → "我的风险真的分散了吗？"
    ↓
记录反思 → 写下发现和下一步行动
```

## 核心理念

这个工具不会告诉你买什么、卖什么。它做的事情是：

1. **让你看见** — 把模糊的感觉变成具体的数字
2. **让你验证** — 把"我觉得"变成"数据说"
3. **让你反思** — 把无意识的操作变成有流程的决策

散户亏钱最大的原因不是选股差，而是没有流程——随手买、恐慌卖、从不复盘。QuantKit 帮你建立这个流程。
